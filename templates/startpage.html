Local Image API.

This API can be used to resize, add filters, crop, extended to integrate in any web application.

Features

  • Upload
  • Browse
  • Manupulate a single image

Setup

Basic required configuration
In the config.php file in root folder edit the following: 


$config['upload_path']['deafult'] 		# Defines where your media images will be stored
$config['cache_dir'] = '.cache' 		# Defines where your cached media images sizes will be stored

Atleast one image size with the following format
In the config.php file in root folder edit the following: 

$config['image_size'][type] = array(
	'width'=>'200',
	'height'=>'200', 
	'aspectratio'=>true,			# Maintain aspect ratio or strech
	'resize' => 'crop',  			# How image will be resized. Either [ crop, landscape, auto, portrait, exact ]
	'label'=>'Thumbnail 200 * 200'		# User Interface Label
);

Use with existing projects

To use this image API in your local project just make sure you have a way to interact with your project database and create hooks in the plugins folder e.g to use an image on upload:

plugins/onupload/your_code_file.php
function hook_your_code_file($url, $data){
	/**
	 * Your code here
	 * 
	 * @param url This is the location of the uploaded image
	 * @param data This is the sent variables $_REQUEST from the broswer on upload
	 */
}
